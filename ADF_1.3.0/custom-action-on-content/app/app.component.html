Help

This is an example of ADF application defining custom actions for Alfresco Content Services.

The application assumes you have an Alfresco Content Services up an running.

Access to the repository explorer and select the right menu (the three vertical dots) and you'll see an action to select. The action differs between folders and content.

If you click on the content action, the Share ID will be created for the content and it will be shown into the document list.

If you click on the folder action, a dummy action is executed.
